## 🖱 담당한 기능 및 업무

- **카드 Viewer페이지**
    - 위젯, 화면, 파일 등 설계된대로 카드 컴포넌트 구현
    - 데이터 저장 및 불러오기 기능
    - 삭제, 즐겨찾기, 필터, 검색, 정렬(시간순, 이름순) 기능 구현
    - 썸네일, 제목 등 속성 정보 변경 기능
- **3D-builder : 설계 및 개발**
    - 객체 배치 및 기즈모 활성화
    - 자식, 부모 객체 연결 기능
- **데이터 연동 및 설계**
    - 데이터 연동 방식 설계

## 💎 성과:
